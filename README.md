# Nhận Diện Phương Tiện Giao Thông (Vehicle Type Recognition)

Dự án ứng dụng CNN (ResNet-50) và các bộ lọc không gian để phân loại phương tiện giao thông.

## Hướng dẫn cài đặt
1. Cài đặt thư viện: `pip install -r requirements.txt`
2. link dataset: https://drive.google.com/file/d/1pNmm9RjcdTJVRl8_uv-Cs5-CahkROKHs/view?usp=sharing
3. Chạy giao diện: `streamlit run app.py`
